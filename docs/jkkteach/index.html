Online Resources for Your Teaching and Learning:

  • Bayesian data analysis programs, at book web site.
  • Excel workbooks that generate sampling distributions from arbitrary populations for any sample statistic. (Of course, you should be doing Bayesian data analysis, not computing p values.)
  • Model exercises. In each of the exercises you generate data in a quick experiment and then explore whether a given model can account for your data. The experiments take 10 minutes or less, and the modeling is done entirely with pencil and paper (and maybe a calculator in a few spots). [These are very old and probably won't work anymore because the server has changed the software it uses.]
