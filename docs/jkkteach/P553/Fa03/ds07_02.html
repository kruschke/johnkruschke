SPSS applied to Lockhart data set DS07_02.DAT
Compare with Worked Example 1 on pp. 332-333

This web page shows an SPSS analysis comparable to Worked Example 1 on Lockhart pp. 332-333, using data set DS07_02.DAT. The SPSS output, and its relation to the Lockhart analysis, will be discussed in class. All the Lockhart data are available on the web. UITS has web documentation about SPSS.

The SPSS Program: 

set highres off .
set width = 80 .
set length = 54 .
data list file = ds07_02.dat free
  / group latency .
variable labels group "Tranquilizer Type" .
value labels group  1 "Control"  2 "T1"   3 "T2".
variable labels latency "Response Latency" .

examine variables = latency by group 
  / plot = boxplot stemleaf histogram npplot .

oneway latency by group(1,3) 
 / format = labels
 / statistics = descriptives homogeneity
 / contrast -1  1  0
 / contrast -1  0  1
 / contrast  0 -1  1
 / ranges = lsd
 / ranges = tukey .

finish .
