  1. (15 pts):

    (a) In a distribution skewed to the right, which is larger, the median or the mean? Why?

    The mean is at least as large as (and typically larger than) the median in a distribution skewed to the right. This is because the mean, which minimizes squared deviations instead of absolute deviations, is more sensitive to outliers than the median.

    (b) Suppose a sample contains the following nine scores: 1, 2, 2, 3, 3, 4, 4, 4, 4. What are the mode, median and mean values?

    The mode is 4. The median is 3. The mean is 27/9=3.0.

    (c) State the definitions of mode, median and mean in terms of the measures of variability that each minimizes. For the sample in Part (b), above, compute the value of the minimal variability for each measure of central tendency.

    The mode minimizes V=(1/N)*SUMidist(xi,M), where dist(xi,M)=0 if |xi-M|=0 and dist(xi,M)=1 otherwise. In this case, V=(1/9)*[dist(1,4) + dist(2,4) + dist(2,4) + dist(3,4) + dist(3,4) + dist(4,4) + dist(4,4) + dist(4,4) + dist(4,4)] = (1/9)*5 = 5/9.

    The median minimizes V=(1/N)*SUMi|xi-M|. In this case, V=(1/9)*[|1-3| + |2-3| + |2-3| + |3-3| + |3-3| + |4-3| + |4-3| + |4-3| + |4-3|] = 8/9.

    The mean minimizes V=(1/N)*SUMi|xi-M|2. In this case, V=(1/9)*[|1-3|2 + |2-3|2 + |2-3|2 + |3-3|2 + |3-3|2 + |4-3|2 + |4-3|2 + |4-3|2 + |4-3|2] = 10/9.

  2. (20 pts):

    (a) For a given population distribution, what does the sampling distribution of the mean look like when the sample size is 1? (describe in words)

    When the sample size is 1, the mean of the sample equals the single score in the sample, and the sampling distribution looks exactly like the population distribution.

    (b) What does the sampling distribution of the mean look like when the sample size approaches infinity? (describe in words)

    The sampling distribution approaches a single peak at the population mean.

    (c) Briefly state the Central Limit Theorem for the sampling distribution of the sample mean.

    The sampling distribution of the mean becomes approximately normally distributed as the sample size increases.

    (d) Suppose we have a population with mean mu = 47.0 and standard deviation sigma = 30.0. If we sample from that population with sample size N = 25, what is the expected value of the sample mean? What is the standard deviation (standard error) of the sampling distribution of the mean?

    The expected value of the sample mean equals the population mean, which in this case is 47.0.

    The standard error of the sampling distribution is sigma/sqrt(N) = 30.0/sqrt(25) = 6.0.

  3. (35 pts):

    Suppose we have a null hypothesis that mu0 = 0.0 and sigma0 = 10.0, and an alternative hypothesis that mu1 = 5.0 (with the same sigma). We assume that the populations are normally distributed.

    (a) What is the effect size for these hypotheses?

    The effect size is (mu1-mu0)/sigma = (5.0-0.0)/10.0 = 0.5.

    (b) Using a Type I error rate of alpha = .05, and a sample size of N = 10, what is the power of the test? (Assume a non-directional, two-tailed test.)

    From the null hypothesis, we compute the critical value for the mean. First consider the "high tail" of the distribution:
    z0(Mcrit) = 1.96 = (Mcrit-mu0)/(sigma/sqrt(N)) = (Mcrit-0.0)/(10.0/sqrt(10))
    which, when rearranged, implies that
    Mcrit = 1.96 * 10.0 / sqrt(10) = 6.20.

    Next, we determine where that critical value lies in the alternative hypothesis sampling distribution:
    z1(Mcrit) = (Mcrit-mu1)/(sigma/sqrt(N)) = (6.20-5.0)/(10.0/sqrt(10)) = 0.38.
    From the standardized normal tables, we find that a Z of 0.38 implies that the power is 35.2%.

    To be completely accurate, we should also determine the contribution to the power from the "low tail" of the distribution. This contribution will be tiny, however, so we'll ignore it here. (But your answer should at least mention the low tail, as it is mentioned here.)

    (c) If we wanted the power to be at least .88, what minimal sample size would we need?

    First, we use the desired power and the alternative hypothesis sampling distribution to determine one constraint on N. From the standardized normal table, we find that the desired power of .88 implies that the critical mean should have a Z value of -1.18. Hence
    z1(Mcrit) = -1.18 = (Mcrit-mu1)/(sigma/sqrt(N)) = (Mcrit-5.0)/(10.0/sqrt(N)).
    Rearranging that equation implies
    Mcrit = -1.18 * 10.0 / sqrt(N) + 5.0.

    From the null hypothesis, have
    z0(Mcrit) = 1.96 = (Mcrit-mu0)/(sigma/sqrt(N)) = (Mcrit-0.0)/(10.0/sqrt(N)).
    Substituting for Mcrit the expression derived above yields:
    1.96 = ( [-1.18 *10.0/sqrt(N)+5.0] -0.0)/(10.0/sqrt(N)).
    Solving for N then yields
    (1.96 - -1.18) * 10.0 / 5.0 = sqrt(N).
    Hence N=39.44, i.e., we need a sample size of at least 40.

    (d) We draw a random sample from the world, with N = 4, and obtain the following scores: 0, 4, 8, 12. Can we reject the null hypothesis? (The null hypothesis was stated at the beginning of this exercise. Assume a non-directional test with alpha = .05.)

    The sample mean is 6.0, so z = (6.0-0.0)/(10.0/sqrt(4)) = 6/5 = 1.20. This does not exceed the critical z value of 1.96, so we cannot reject the null hypothesis at the .05 level.

    (e) Using the standard deviation assumed by the null hypothesis, compute the 95% confidence interval for the population mean, based on the sample mean from part (d). Assume a non-directional, two-sided confidence interval.

    The confidence interval is the sample mean, 6.0, plus or minus 1.96 standard errors:
    Low end: 6.0 - 1.96*10.0/sqrt(4) = -3.8.
    High end: 6.0 + 1.96*10.0/sqrt(4) = 15.8.

    (a) Can the null hypothesis be rejected? (Assume alpha = .05, two-tailed.)

    The mean change in subjective comfort is an increase of 0.60 (from 4.90 to 5.50). The estimated sigma is sqrt(SS/df) = 1.174. Then t = 0.60/(1.174/sqrt(10)) = 1.62.

    From the t-table, we find that tcrit(9) = 2.262 (for a two-tailed test with Type I error rate of .05). Hence we cannot reject the null hypothesis of no change.

    Upper bound of different between means: 0.60 + 2.262 * 1.174/sqrt(10) = 1.44.

    The estimated effect size is 0.60/1.174 = 0.51.
