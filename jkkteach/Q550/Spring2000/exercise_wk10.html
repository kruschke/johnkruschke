  • (Ch. 6.) Consider a competitive learning network with two input nodes and two output nodes. Suppose there are three input patterns to learn: (.9,.6), (.9,.4), (.1,.5). Suppose that the weight vectors of the two output nodes start at (.6,.6) and (.4,.4). Let the learning rate be 0.5. Compute the values of the weights for three epochs through the three input patterns (in the fixed order listed before). Plot the patterns and the trajectories of the weight vectors like the graphs in Figure 6.4.
    Compute the activation of the cluster nodes in terms of Euclidean distance, NOT as weighted summation. That is, do NOT do it as in Equation 6.1 of the text book!

    For example, the exercise states that the two cluster nodes start of with weights of (.6,.6) and (.4,.4). The first input pattern is (.9,.6). Therefore the distance between the first cluster node and the input pattern is

    [(.6-.9)^2 + (.6-.6)^2]^.5 = 0.30
    and the distance between the second cluster node and the input pattern is
    [(.4-.9)^2 + (.4-.6)^2]^.5 = 0.54
    The first cluster node is closer to the input pattern, so it is the winner and it changes its weights. The exercise states that the learning rate is 0.5, so the changes in weights for the winning (first) cluster node are
    change in w1 = 0.5 * (.9-.6) = 0.15
    change in w2 = 0.5 * (.6-.6) = 0.00
    Hence the new weights for the first cluster node are
    w1 = .6 + .15 = .75
    w2 = .6 + .00 = .60

    Continue in this manner through all the training patterns, for three epochs.
