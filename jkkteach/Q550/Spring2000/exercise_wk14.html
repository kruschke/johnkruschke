Evolution of networks. Consider the XOR problem in a 1-hidden layer backprop network (cf. pp. 117-126). In previous homework, you found that success in solving the task depended on the number of hidden nodes. In this exercise, we'll pretend that networks that learn the XOR pass on their genes to the next generation, and the genes determine the number of hidden nodes in the progeny. That is, a parent network with H hidden nodes has children with H hidden nodes.

Let's suppose that if a network gets below an error of 0.1 by sweep 4000 (puberty), then the network generates 2 children, with the same number of hidden nodes (but with random initial weights).

There is a cost of having hidden nodes, however, because it takes more food to maintain a larger network. So, while larger networks will probably solve the learning task better than smaller networks, they might not find enough food to live until puberty (4000 sweeps).
