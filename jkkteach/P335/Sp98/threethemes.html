Three themes of memory

  1. An event can be encoded and processed different ways. We remember our encoding of the event, not the external event itself.

      Examples:
    • Depth of processing experiments
    • Jacoby '83 experiment on implicit memory and indirect tests
    • Transfer appropriate processing
    • ...

  2. Memory associates events with their external context and our internal state.

      Examples:
    • Encoding specificity experiments
    • The cognitive interview
    • Context-bound imagery
    • ...

  3. Memory is reconstructive: It uses context and knowledge structures (schemas) to reconstruct the seemingly most likely event.

      Examples:
    • Fabricated memories
    • ...
