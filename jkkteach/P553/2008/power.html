Statistical Power and Effect Size

Figure 1 (below) illustrates the concepts of statistical power and effect size. The diagram will be explained in detail in lecture. One point to be emphasized is that power lives in the sampling distributions, whereas effect size lives in the population distributions.

Figure 1. Illustration of effect size and power.
The "1.65" in the sampling distribution of the null
hypothesis assumes a Type I error rate of 5%.


Figure 2. Sampling distributions
with critical value of sample mean
explicitly marked.

All the basic computations regarding power result from just the two equations that describe the position of the critical value in the sampling distributions. Figure 2 (right) shows the two sampling distributions with the critical value of the sample mean labeled as Mcrit. Recall from Figure 1 that σM = σ/sqrt(N).

In the null hypothesis sampling distribution, the diagram indicates that
znull(Mcrit) = ( Mcrit − μnull ) / σM [Eqn. 1]
The value of the Type I error rate sets the desired value of z; if we want the Type I error rate to be .05, then z(Mcrit) = 1.65, as shown in the diagram.

In the alternative hypothesis sampling distribution, the diagram indicates that
zalt(Mcrit) = ( Mcrit − μalt ) / σM [Eqn. 2]
From this value of zalt(Mcrit), we can determine the power, or, if given a desired power, we can determine the needed value of zalt(Mcrit).

Determining power: If you are given μnull, μalt, σ, alpha (i.e., the Type I error rate), and N, then you can compute the power from the two equations. Start with Equation 1 to determine Mcrit. Then use this value in Equation 2 to determine the resulting power.

Example. Suppose μnull=0, σ=10, μalt=5, alpha=.05, and N=16. What is the power?

Before jumping into the equations, it would help if you entered these values into a diagram like Figure 2, above.

We start with the null hypothesis to determine Mcrit. Because we want alpha=.05 one-tailed, we need znull(Mcrit)=1.65. It's positive 1.65, instead of negative 1.65, because μalt is greater than μnull. Also notice that because σ=10 and N=16, σM = 10/sqrt(16) = 2.5. We then plug in the known numerical values into Equation 1, znull(Mcrit) = ( Mcrit − μnull ) / σM, which becomes 1.65 = ( Mcrit − 0 ) / 2.5. Solving for Mcrit yields Mcrit = 2.5*1.65.

We then consider the alternative hypothesis. Using Equation 2, we plug in the value of Mcrit that we just determined. zalt(Mcrit) = ( Mcrit − μalt ) / σM = ( 2.5*1.65 − 5 ) / 2.5 = −0.35. We convert this z value to a probability value by looking up the probabilities in a normal table, such as Table A-1, pp.639-641 of Aron and Aron. There we find that for z=0.35, the percentage from the mean is 13.68. Because our Mcrit is below the mean (that's why the z value was negative), the power is 13.68 plus 50, i.e., 63.68.

Determining sample size: If you are given μnull, μalt, σ, alpha (i.e., the Type I error rate), and the desired power, then you can compute the needed N from the two equations. Essentially, you simply plug in all the given information into the two equations, which yields two equations with two unknowns; the unknowns are N and Mcrit. Solving the two equations yields N.

Example. Suppose μnull=0, σ=10, μalt=5, alpha=.05, and a desired power of .75. What is the minimal N needed?

Before jumping into the equations, it would help if you entered these values into a diagram like Figure 2, above. Also, notice the similarity of this example to the previous example: The means and standard deviations are the same. We found that for N=16 the power was .6368; now we want a power of .75, so we know that N will need to be bigger than 16.

We start with the null hypothesis to describe Mcrit. Because we want alpha=.05 one-tailed, we need znull(Mcrit)=1.65. It's positive 1.65, instead of negative 1.65, because μalt is greater than μnull. We hypothesize σ=10 but we don't know N, so all we can say is σM = 10/sqrt(N). We then plug in the other known numerical values into Equation 1, znull(Mcrit) = ( Mcrit − μnull ) / σM, which becomes 1.65 = ( Mcrit − 0 ) / (10/sqrt(N)), which rearranged becomes 1.65 * 10 / Mcrit = sqrt(N).

We then consider the alternative hypothesis. We want the power to be .75, which entails that Mcrit lies below μalt such that the area between μalt and Mcrit is .25. From a table of normal probabilities, such as Table A-1 in Aron and Aron, we find that a z value of −0.68 is needed to get at least .25. Using Equation 2, zalt(Mcrit) = ( Mcrit − μalt ) / σM, we get −0.68 = (Mcrit−5)/(10/sqrt(N)).

The previous two paragraphs have just yielded two equations with two unknowns, namely N and Mcrit. Now we just solve the equations. The first equation gave us a forμla for sqrt(N), which we plug into the second equation: −0.68 = (Mcrit−5)/(10/sqrt(N)) = (Mcrit−5)/(10/[1.65 * 10 / Mcrit]). Rearranging to solve for Mcrit yields Mcrit=3.54. Now we can plug that into the first equation, sqrt(N) = 1.65*10/3.54, hence N=21.7. That is, we need N=22 to get a power of at least .75.

Doing this in the general case yields:
N = [ ( znull(Mcrit) − zalt(Mcrit) ) / effect size ]2 [Eqn. 3]
For example, if we have a Type I error rate of .05, then znull(Mcrit) = 1.65. If we want a power of 0.84, then zalt(Mcrit) = −1.00 (notice the negative sign, because Mcrit is less than μalt). For a moderate effect size of 0.5, then Equation 3 implies that N = [ ( 1.65 − −1.00 ) / 0.5 ]2 = 28.09, so we need N=29 to achieve a power of at least 0.84.
