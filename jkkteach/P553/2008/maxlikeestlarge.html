Modeling with the normal distribution:
Maximal likelihood estimation of its parameter values

The normal (a.k.a. Gaussian) distribution is the well-known bell-shaped curve. The central limit theorem is one motivation for thinking of the normal distribution as the canonical model of a population. The normal probability density of variable x, given parameter values mu and sigma, is
p( x | mu,sigma ) =
1
SQRT(2*PI*sigma2)
 EXP( -0.5 [ (x-mu) / sigma ]2 )

The question is now this: Given a set of observed scores x1,...,xN, what parameter values mu and sigma yield the highest probability of those scores? For example, suppose we have three scores, with x1=10, x2=20 and x3=30. If mu=1000 and sigma=1, then those three scores are extremely improbable. If mu=20 and sigma=10, however, then the three scores are more probable. What values of mu and sigma yield the highest probability for the scores? These parameter values provide the "best fit" of the model to the data, in the sense that these parameter values yield the highest proabality of the sample data, relative to all other possible parameter values.

We assume that the scores are generated independently. This implies that the probability of getting the combination of all N scores is the product of their individual probabilities:

p( x1 and ... and xN | mu,sigma )
= p( x1 | mu,sigma ) * ... * p( xN | mu,sigma )

We want to determine the values of mu and sigma that maximize that probability. A standard method from calculus, for determining the maximum of a function, is computing the derivative of the function and then solving for the value that makes the derivative zero. In principle, we could apply this method directly to the formula above, but in practice, computing the derivative of an indexed PROD gets messy. So, we will first convert the PROD to a SUM by applying the logarithm function to both sides of the equation. The logarithm is a monotonic (order-preserving) continuous function, so if we find the values of mu and sigma that maximize the log of the probability, those same values maximize the probability itself. In summary, we started with a set of scores, x1,...,xN. We decided to try to model this set of scores as being generated by a normal probability distribution. We then found the values of the parameters mu and sigma that made the distribution best fit the scores; that is, we found the values of mu and sigma that maximized the likelihood of the scores. The resulting values turned out to be familiar: The maximal likelihood estimate of mu is the mean of the scores, and the maximal likelihood estimate of sigma2 is the variance of the scores.
