8. (4 pts.) Consider two population distributions. We are told that mu_0 = 100, mu_1 = 95, and sigma = 15 (for both). What is the sample size needed for a power of .8 or better? Here's the solution:

Notice that mu_1 is LESS than mu_0, so you can draw the picture of the sampling distributions correctly. Now go through the usual two steps in computering power. First, from the null hypothesis,
z_crit = -1.65 = ( M_crit - 100 ) / ( 15 / sqrt(N) )
Second, from the alternative hypothesis,
z for .8 = 0.84 = ( M_crit - 95 ) / ( 15 / sqrt(N) )
That gives two equations with two unknowns. Using the first equation, we get
M_crit = -1.65 * ( 15 / sqrt(N) ) + 100
Plugging that into the second equation yields
0.84 = ( [ -1.65 * ( 15 / sqrt(N) ) + 100] - 95 ) / ( 15 / sqrt(N) )
Solving for N yields
N = [ ( 1.65*15 + .84*15 ) / (100-95) ]^2 = 55.8
So, we need N to be at least 56 for a power at least 0.8. (If I made an error anywhere in there, please let me know! You can check it by computing the power for N=56.)
