SPSS applied to Lockhart data set DS03_01.DAT

The Study Guide, pp. 117-122, describes how to use SPSS for Windows, applied to Lockhart data set DS03_01.DAT. This web page shows how to do a very similar application using good old-fashioned command-line SPSS.

SPSS program:
The program lines are not commented here; they will be discussed in class. 

set highres off .
set width = 80 .
set length = 54 .

data list file = ds03_01.dat free
  / group anxiety .

variable labels group "Therapy type" .
value labels group  1 "Control"  2 "Desensitization" .

variable labels anxiety "Increase in Heart Rate" .

examine variables = anxiety by group 
  / plot = boxplot stemleaf histogram npplot .

t-test groups = group 
  / variables = anxiety .

finish .

SPSS output:
The above program was run on the Unix computer named Zinc, using the command
spss -m < ds03_01.sps > ds03_01.lst
The file, ds03_01.sps, contains the spss commands above. The file, ds03_01.lst is the output file, shown below.
