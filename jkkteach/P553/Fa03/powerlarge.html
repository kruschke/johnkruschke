Statistical Power and Effect Size

Figure 1 (below) illustrates the concepts of statistical power and effect size. The diagram will be explained in detail in lecture. One point to be emphasized is that power lives in the sampling distributions, whereas effect size lives in the population distributions.

Figure 1. Illustration of effect size and power.
The "1.65" in the sampling distribution of the null
hypothesis assumes a Type I error rate of 5%.


Figure 2. Sampling distributions
with critical value of sample mean
explicitly marked.

All the basic computations regarding power result from just the two equations that describe the position of the critical value in the sampling distributions. Figure 2 (right) shows the two sampling distributions with the critical value of the sample mean labeled as Mcrit. Recall from Figure 1 that sigmaM = sigma/sqrt(N).

In the null hypothesis sampling distribution, the diagram indicates that
znull(Mcrit) = ( Mcrit - munull ) / sigmaM [Eqn. 1]
The value of the Type I error rate sets the desired value of z; if we want the Type I error rate to be .05, then z(Mcrit) = 1.65, as shown in the diagram.

In the alternative hypothesis sampling distribution, the diagram indicates that
zalt(Mcrit) = ( Mcrit - mualt ) / sigmaM [Eqn. 2]
From this value of zalt(Mcrit), we can determine the power, or, if given a desired power, we can determine the needed value of zalt(Mcrit).

Determining power: If you are given munull, mualt, sigma, alpha (i.e., the Type I error rate), and N, then you can compute the power from the two equations. Start with Equation 1 to determine Mcrit. Then use this value in Equation 2 to determine the resulting power.

Determining sample size: If you are given munull, mualt, sigma, alpha (i.e., the Type I error rate), and the desired power, then you can compute the needed N from the two equations. Essentially, you simply plug in all the given information into the two equations, which yields two equations with two unknowns; the unknowns are N and Mcrit. Solving the two equations yields N. Doing this in the general case yields:
N = [ ( znull(Mcrit) - zalt(Mcrit) ) / effect size ]2 [Eqn. 3]
For example, if we have a Type I error rate of .05, then znull(Mcrit) = 1.65. If we want a power of 0.84, then zalt(Mcrit) = -1.00 (notice the negative sign, because Mcrit is less than mualt). For a moderate effect size of 0.5, then Equation 3 implies that N = [ ( 1.65 - -1.00 ) / 0.5 ]2 = 28.09, so we need N=29 to achieve a power of at least 0.84.
